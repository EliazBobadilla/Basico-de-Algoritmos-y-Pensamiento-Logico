# ¿Qué es el sistema binario?
Es un sistema de numeración que está representado por uno (1) y cero (0)

El 1 y 0 representan diferentes valores:
- Hay corriente = 1
- No hay corriente = 0
- Positivo (+) = 1
- Negativo (-) = 0
- True = 1
- False = 0
